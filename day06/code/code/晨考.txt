1.方法的四种形式的语法格式
	无参无返回值的方法
	public static void test () {}
	有参无返回值的方法
	public static void test (int a, int b) {}
	无参有返回值的方法
	public static int test () {
		return 3;
	} 
	有参有返回值的方法
	public static String test (String a) {
		return "嘻嘻";
	}
