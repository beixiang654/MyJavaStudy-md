## 第三天

```java
1、算术运算符:+-*/
2、关系运算符:> < >= <= !=（非等于） ==
3、逻辑运算符:
	短路与:&& 语法格式：关系运算符表达式1 &&  关系运算符表达式2 && 关系运算符表达式3 &&...
	只要有一个结果为false整体就为false,并且不往下继续判断表达式
	
	短路或:|| 语法格式：条件1 || 条件2  ||条件3 || ...
	只要有一个结果为true整体就为true,并且不往下继续判断表达式
	
	非:! 语法格式：!条件
	结果取反
4、分支结构:if(表达式){
语句
}
如果满足if后面的表达式则执行大括号里面的语句
if(表达式){
语句1
}
else{
语句2
}
如果满足if后面的表达式则执行语句1，不满足则执行语句2
if(表达式1){
语句1
}
else if(表达式2){
语句2
}
else if(表达式3){
语句3
}
else{
语句4
}
如果满足表达式1则执行语句1，并且不往下执行，如果不满足则接着往下判断表达式2，满足则执行语句2，否则往下执行都不满足则执行语句4
5、switch(变量)也是一种选择，只有当满足当前的条件的时候才会执行前的语句体
{
case 常量1:
	语句体1;
	break;
case 常量2:
	语句体2;
	break;
default:
	语句体3;
	break
	
}
6、还扩展了scanner类，可以获取从控制台输入的数
import java.util.Scanner;//先声明

Scanner 变量名1 = new Scanner(System.in);
int 变量名2 = 变量名1.nextInt();
```

