1.三目运算符语法格式和执行流程
	语法格式:
		数据类型 x = 表达式 ？ 值1:  值2;
	执行流程: 
		首先判断表达式 ,如果为true ，就会将值1赋值给x。
		如果为false 就将值2赋值给x
2.while循环的语法格式和执行流程
	while (布尔表达式) {
		循环体
	}
	执行流程:   会先判断布尔表达式，如果i布表达式true
	执行循环体中的代码。再次回到while判断布尔表达式
	如果布尔表达式还是true，执行循环体。直到布尔表达式为
	false的话，不再执行循环体，循环结束
3.do-while循环的语法格式和执行流程【开发中不用】
	do {
		循环体;
	} while (布尔表达式);
	执行流程:  代码执行到do之后，会先执行循环体。
	然后再进入while去判断布尔表达式，如果为true ，再回到
	do 再次执行循环体。直到while后面的布尔表达式为false的话
	终止循环。
4.for循环的语法格式和执行流程
	for (表达式1; 表达式2; 表达式3) {
		循环体;
	}
	执行流程:  代码碰到for以后，先执行表达式1,然后
	再判断表达式2,如果表达式2为true，执行循环体。
	执行表达式3。再次判断表达式2，执行循环体。
	再次执行表达式3。再次判断表达式2，直到表达式2为false
	则终止循环
5.打印九九乘法表
	for (int i = 1; i < 10; i++) {//行
		for (int j = 1; j <= i; j++) {//列
			System.out.print(j + "*" + i +"=" + j*i + "\t");
		}
		System.out.println();
	}
